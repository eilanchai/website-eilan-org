## One Red Paperclip

Inspired by Kyle MacDonald's one red paperclip project in which he traded a paperclip for progressively more valuable items until he owned a house, I completed a similar series of trades as a project for my negotiations class. Starting with a red paperclip, I traded up through a cat-shaped eraser, a minifridge, a gaming chair, and a GPU, ending with an older iPad Pro and Apple Pencil.

## AWS FinTech Hackathon (2023)

As part of a team of seven, built a document scanner using Amazon Textract that reads purchase orders and automatically fills forms with the extracted data. Placed second in Amazon's internal 2023 AWS FinTech Hackathon.

## Live Cloud Data Pipeline

A practice project connecting Microsoft Access to live data in Azure, with updates pushed to Tableau. Built to learn real-time cloud data connections.
